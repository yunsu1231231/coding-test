### 트리는 결국 그래프에서 사이클이 없는 것 결국 단방향이 아니면 중복처리 + 순회 방식은 똑같다

### 트리 
관계 중심 -> 인접리스트  
구조 중심 -> Node 클래스  
문제에서 트리 형태가 주어지면 -> Node 클래스  
문제에서 단순히 연결된 간선 -> 인접리스트  

정의: 트리란 사이클이 없고, 모든 노드가 단 하나의 경로만 연결되는 계층적 구조.  
부모 -> 자식이 존재하고, 루트로부터 모든 노드가 이어지는 연결 구조
